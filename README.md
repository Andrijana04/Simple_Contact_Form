This is a simple contact form built using **HTML, CSS, and JavaScript**.
---

## Features
- Form validation
- Required fields check
- Email format validation
- Error messages display
- Prevents empty submissions

---

## How it works
- User fills in first name, last name, email, and message
- JavaScript checks if fields are empty
- Email is validated using "@" check
- If everything is valid → success message is shown

---

## Technologies Used
- HTML
- CSS
- JavaScript (form validation + DOM manipulation)

---

## Logic
- `preventDefault()` stops page reload
- Errors are shown using `textContent`
- Simple validation rules are applied
